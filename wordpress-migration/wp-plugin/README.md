# TPS1 Core

Plugin nền để đăng ký custom post type và taxonomy cho WordPress local/staging.

## Có sẵn

- `product`
- `news`
- `knowledge`
- `industry`
- `service`
- `policy`

## Cách dùng

1. Copy thư mục `wp-plugin` vào `wp-content/plugins/tps1-core`.
2. Activate plugin trong WordPress admin.
3. Vào `Settings -> Permalinks` và bấm Save lại một lần nếu cần.
