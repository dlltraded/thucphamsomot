# Local Checklist

## Bước 1: dựng WordPress

- `docker compose up -d`
- mở `http://localhost:8080`
- hoàn tất cài đặt

## Bước 2: cài theme nhẹ

- cài `GeneratePress` hoặc `Blocksy`
- không cài page builder nặng lúc đầu

## Bước 3: tạo CPT

- `product`
- `news`
- `knowledge`
- `industry`
- `service`
- `policy`

## Bước 4: tạo field

- summary
- sections
- faqs
- featured image
- SEO title / meta description

## Bước 5: import dữ liệu

- copy nội dung từ source hiện tại
- ưu tiên migrate bài viết và ảnh trước
- kiểm mobile trên iPhone / Android viewport

## Bước 6: test trước khi public

- home
- product detail
- news detail
- knowledge detail
- quote form
- mobile menu
